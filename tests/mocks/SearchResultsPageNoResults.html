Merkliste aktivieren

Durch Anklicken einer Merkliste wird diese an die erste Position platziert und wird zur aktiven Merkliste. Ist das Stern-Icon gefüllt, führt der Klick auf diese aktive Merkliste zur Anzeige der Merkliste.

Trefferliste filtern

So funktioniert das Filtern

Klicken Sie einen Filter an, um dessen Werte anzuzeigen.
Sie können Ihre Trefferliste einschränken, indem Sie einen oder mehrere dieser Werte von einem oder mehrere(n) Filtern auswählen und auf die Schaltfläche "Filter aktivieren/deaktivieren" klicken. Sie können Ihre eingeschränkte Trefferliste wieder erweitern, indem Sie die Häkchen der Werte unter der Rubrik "Aktive Filter" wieder entfernen und die Schaltfläche "Filter aktivieren/deaktivieren" auslösen.
